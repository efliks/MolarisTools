# MolarisTools

A Python library and a set of example scripts to facilitate working with the Molaris program.

* AssignEVB
* AssignEVBSymmetric
* CallGaussian
* CallMopac
* Charges
* CheckMapping
* CompareMopacVSGaussian
* MopacNumericalForces
* PrepareConstraints
* PrepareWHAM
* QMMM
* QMMM\_Alchemical\_FEP
